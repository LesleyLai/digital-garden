The *Kronecker delta* is a function of two variables. The function is 1 if the variables are equal, and 0 otherwise:
$$
\delta_{ij} = \begin{cases}
0 &\text{if } i \neq j,   \\
1 &\text{if } i=j.   \end{cases}
$$
