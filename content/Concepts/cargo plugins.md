# Cargo Plugins
## Cargo-machete
Can be used to cut out unused dependencies. `cargo install cargo-machete` then use `cargo machete`

## Cargo-audit
Detect vulnerable crates

## Cargo-expand
See: [[./cargo expand|cargo expand]]
Show result of macro expansion
