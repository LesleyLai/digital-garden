Three different permission groups
- owner (`u`)
- group (`g`)
- everyone (`o`)

Three permissions
- read (`r`) (octal value 4)
- write (`w`) (octal value 2)
- execute (`x`) (octal value 1)

`chmod` is the command to change permissions.
