**HTTP status code** are 3 digits codes used to indicate whether an HTTP requests has been successfully completed.

Status code are broken into 5 blocks:
- 1xx informational
- 2xx Success
  - 200 OK - most common responses
- 3xx Redirection
- 4xx Client Error
  - 404 NOT FOUND
- 5xx Server Error
